Esse repositório foi criado para fins didádicos a respeito do framework Angular. Nele, utilizei o Angular para desenvenvolver o backend e frontend. Além disso, consumi a API da WeatherMap para retornar os dados do clima, como: temperatura atual e velocidade do vento, temperatura mínima e temperatura máxima.

Olhe o projeto em produção: https://weather-app-angular-17-five.vercel.app/
