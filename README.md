Esse repositório foi criado para fins didádicos a respeito do framework Angular. Nele, utilizei o Angular para desenvenvolver o backend e frontend. Além disso, consumi a API da WeatherMap para retornar os dados do clima como: temperatura atual, velocidade do vento, temperatura mínima e temperatura máxima. Tudo isso a partir da cidade digitada pelo usuário.

Olhe o projeto em produção: https://weather-app-angular-17-five.vercel.app/
